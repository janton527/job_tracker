# AI Usage Documentation

## Tools Used
- ChatGPT

## Key Prompts
1. "How would I be able to reuse the same template for Add and Update?"
2. "How would I create the CRUD logic instead of repeating code for each page?"
3. "Why do I keep getting a 400 Bad Request error?"
4. "What would be the best way to implement a job matching feature?"
5. "What would be some good CSS improvements that I could make?"

## What Worked Well
- Good at explaining how to use frameworks/concepts that I am unfamilliar with
- Was able to give good boilerplate code to get the ball rolling
- Very helpful at helping me track down bugs and issues with the syntax
- Many helpful suggestions for what CSS to add to make a professional looking UI

## What I Modified
- Changed variable and function names to better match my code
- Adjusted CSS output to better match what I wanted

## Lessons Learned
- AI is better used to explain concepts than just generate code
- AI can be used to help bridge the gap if you are not familiar with a language or framework
